How This Was Made

I had the funny idea of, "What if someone could both hire and fire you from your resume, just by clicking a few buttons?".

I prompted Claude.AI, which can't compile TypeScript, so I copied it over to a fresh Angular project made with `ng init` and I removed the boilerplate. Initially nothing worked because Clauded create an app expecting AppModule and now the Angular team is pushing the "standalone" model. I switched the config and imports to standalone.

The bots are not able to create entire applications, at best, they can create a section of an application and a human must tie the pieces together.

The CSS is from latex-css, which by default is very narrow, so I overrode the width.

If this was a professional product, I'd add a build script, unittests and package it in a Dockerfile along with nginx. Instead this will be published on Github so that there is a convenient link to it.
